# Shadow Boxing Randomizer

A professional, voice-coached shadow boxing workout randomizer built by **Wim Demeere**.

Live version: [https://wimdemeere.github.io/shadow-boxing/](https://wimdemeere.github.io/shadow-boxing/)

## Features

- Customizable round length, rest time, and number of rounds
- Multiple modes: **Boxing**, **Kickboxing**, **Muay Thai**, and **MMA**
- Single technique or combination mode
- Intelligent combo generation (no technique or same-side repeat more than twice in a row)
- Voice coaching with adjustable call tempo
- 3-2-1 countdown with voice ("begin")
- Screen Wake Lock support (keeps your phone screen on during workouts)
- Fully responsive and works great on mobile

## How to Use

1. Open the app: [https://wimdemeere.github.io/shadow-boxing/](https://wimdemeere.github.io/shadow-boxing/)
2. Configure your session (round length, rest, number of rounds, mode, etc.)
3. Tap **START**
4. Follow the voice calls and technique prompts

## Installation (PWA)

You can install this app on your phone for quick access:

- **Android**: Open in Chrome → tap the menu → **Install app**
- **iOS**: Open in Safari → tap Share → **Add to Home Screen**

## Deployment

This is a static single-file web app. You can host it on:

- GitHub Pages (current hosting)
- Netlify
- Vercel
- Any static hosting service

Just upload the `index.html` file (and optionally the `manifest.json` + icons for PWA support).
